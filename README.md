# Clerk's Homebrew Tap

Homebrew formulae for [Clerk](https://clerk.com) tools.

## Installation

```bash
brew install clerk/stable/clerk
```

### Installing a specific major version

```bash
brew install clerk/stable/clerk@1
```

Versioned formulas are keg-only. To link a versioned formula into your PATH:

```bash
brew link clerk@1
```

## Updating

```bash
brew upgrade clerk
```
